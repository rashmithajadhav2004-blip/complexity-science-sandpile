# complexity-science-sandpile
# Self-Organized Criticality in a 2D Sandpile Model

This project studies a 2D sandpile as a complex system and analyzes avalanche dynamics, connectivity, and self-organized criticality.

## Author
Rashmitha G

## Files
- `report.tex` — LaTeX manuscript
- `project_report.pdf` — compiled report
- `sandpile_soc.py` — Python simulation code
- `sandpile_outputs/` — generated figures, table, and data

## Results
The simulation compares 4-neighbor and 8-neighbor connectivity and generates:
- avalanche size-frequency distribution
- connectivity vs mean avalanche size
- summary statistics for the report

```bash
python sandpile_soc.py
